# 选题：On-chain 超休闲小游戏 + 实时排行榜 + 微量竞猜

---

## Research：为什么 Monad 更适合这个场景（~300字）

### 场景定义

一个超休闲点击游戏（类似 Flappy Bird / 跳一跳的 On-chain 版）：用户每局支付 0.001 MON 入场，通过实时操作获得分数，分数即时上链更新排行榜，每局结束后奖池按排名分配给前 10% 玩家

### 为什么需要高频交互

这种游戏模式下，单局会产生多次链上交互——游戏开始（Stake 锁定）、每 N 次操作更新分数状态（Snapshot）、游戏结束（结算）。按平均一局 20-30 次操作、玩家每天打 20 局计算，单个活跃用户每天产生超过 400 笔交易。如果有 1000 个日活用户，日均链上交易量就是 40 万笔——这个频率远超 DeFi 或 NFT 的场景

### 如果链很慢或手续费很高

- **T + 12 秒确认**（以太坊主网）：用户点完最后一帧游戏已经结束了 12 秒分还没出来，游戏节奏被彻底打断
- **Gas 费 $0.10-0.50**（以太坊主网 / 部分 L2）：入场费才 $0.001，每次操作花 $0.10 等于每局亏 100 倍，经济模型直接崩溃
- **Sequencer 延迟**（OP Rollup）：虽然便宜但有出块延迟，排行榜无法实时更新，失去了「越玩越上头」的竞技感

### Monad 带来的帮助

Monad 的 400ms 出块 + 并行执行架构直接解决上述问题：交易在 1 秒内确认，Gas 费几乎为零，10,000 TPS 的吞吐量足够支撑千级 DAU 的实时交互。EVM 兼容意味着 Solidity 直迁，钱包工具（MetaMask/Phantom）即插即用

### 是否真的需要链上记录

**需要。** 排行榜的公正性是这个产品的核心信任基础——如果跑在中心化数据库里，运营方可以随意改分、插队、暗箱操作。链上排行榜可以让任何人验证每局分数的生成过程，奖池分发通过智能合约自动执行，不需要信任运营方。此外，NFT 成就徽章、Referral 追踪等功能天然需要链上可组合性

## Tech：Demo 功能清单

### 核心合约模块

| 模块 | 功能 | 交互频率 |
|---|---|---|
| GameSession | 创建局、Stake 锁定、结算 | 每次游戏启动/结束 |
| ScoreOracle | 接收客户端签名分数、验证防作弊 | 每局 N 次 Snapshots |
| Leaderboard | 全局 + 周榜实时更新 | 每个新分数触发重排 |
| PrizePool | 累积奖池、按排名分配 | 每分钟结算（动态） |
| BadgeNFT | 每周 Top10 徽章 | 每周 1 次 Mint |

### 前端功能清单

- 一键开局：Wallet Connect → Stake 0.001 MON → Start
- 实时操作：每 3-5 次操作发送一次 Hash 上链（平衡频率和成本）
- 排行榜面板：实时滚动更新，显示 Top100 + 你的排名
- 局内回放：链上 GameSession 数据可以回放验证
- 防作弊：分数变化曲线自校验 + 反女巫验证（NFT 持有者免验证）

### 链下辅助

- 轻量级 Indexer：监听链上事件，实时推送排行榜给前端（Monad 400ms 出块下不需要 indexer 也能接近实时，但推荐用 Indexer 做排行榜去重排序）
- Session Cache：短时间内的 Snapshot 先在本地聚合，每 5 次操作批量提交

## Ops：3 个适合 Meme 或社区传播的切入点

### 1. 😂「链上菜鸡鉴定器」

> "你在以太坊上玩一局 Flappy Bird 花掉 $50 Gas，分数只有 3 分。在 Monad 上花 $0.001，还是 3 分。
>
> 链换了菜鸡还是你"

以 Gas 费对比制造反差幽默——「省了 $49.99 但分数没变」这个 Self-deprecating 的笑点适合 X/抖音传播。配合一个 Gas 费用对比的截图 Meme，天然 viral

### 2. 🏆「每周 Top10 上链挂墙」

> "你上的不是榜，是一条永不可篡改的尊严"

每周排行榜前 10 名获得一个独一无二的 Badge NFT（链上可查、不可篡改、带有周次编号）。用户会自发在 X 上晒徽章，形成每周固定传播周期。早期稀缺性——前 10 周 Top10 的 NFT 被称为「Genesis Badge」，持有者在后续赛季享受手续费减免

### 3. 🎮「用 ETH 玩 vs 用 MON 玩」对比挑战

> "同样的合约，同样的游戏，两条链上跑。ETH 版一局 $0.50 Gas，Monad 版 $0.001。ETH 版等 12 秒结算，Monad 版秒到。
>
> 你来选"

策划一个社区挑战赛——部署同一套游戏合约到 ETH Sepolia 和 Monad，让用户亲身体验延迟和成本的差异，截图对比发推，带话题 #MonadFeelsLikeTheFuture。EVM 兼容的好处在这里变成了最直接的传播素材：代码一样，体验天差地别
